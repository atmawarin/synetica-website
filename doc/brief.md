## Technical Considerations

### Platform Requirements

- **Target Platforms:** Responsive web application optimized for desktop and mobile devices
- **Browser/OS Support:** Modern browsers (Chrome, Firefox, Safari, Edge) with mobile-first responsive design
- **Performance Requirements:** Page load speeds under 3 seconds, optimized for Indonesian internet infrastructure and mobile connections
- **SEO Optimization:** Technical SEO foundation to support case study content strategy and keyword targeting

### Technology Preferences

- **Static Site Generator:** Hugo for ultra-fast performance, excellent SEO, and simple content management via Markdown files
- **Frontend:** Hugo templates powered by Tailwind CSS utility classes and shadcn/ui component primitives for accessible, consistent interactions
- **Theme Baseline:** Start from the Tweakcn theme `cmftuwple000004kv5zzr2ukn`, translating its layout, spacing, and typography tokens into Synetica's brand system with modern geometric sans-serif primary typography and humanist sans-serif secondary typography
- **Content Management:** File-based content using Markdown for case studies, blog posts, and service descriptions - easy to version control and collaborate
- **Form Handling:** Third-party services (Netlify Forms, Formspree) for contact forms and consultation requests
- **Hosting/Infrastructure:** Static hosting (Netlify, Vercel, GitHub Pages) with global CDN - extremely fast and cost-effective

### Architecture Considerations

- **Repository Structure:** Git-based content management with Hugo site structure, simple content folder organization for easy collaboration
- **Service Architecture:** Static site with external service integrations - no complex backend infrastructure needed; Tailwind CLI + PostCSS pipeline runs during Hugo builds to generate purged CSS for shadcn components
- **Integration Requirements:** 
  - WhatsApp integration via JavaScript/URL schemes for consultation requests
  - Google Analytics and Search Console for SEO tracking
  - Email marketing platform integration via form submissions
  - Git-based workflow for case study publishing and version control
- **Security/Compliance:** 
  - HTTPS via hosting platform (automatic with Netlify/Vercel)
  - Minimal data collection - forms handled by third-party services
  - No user data storage on site infrastructure
  - Static site security advantages (no server vulnerabilities)

### Content Management Strategy

- **File-Based Content:** Markdown files for case studies with YAML frontmatter for metadata and SEO optimization
- **Language Structure:** Primary English content with selective Indonesian translations using Hugo's i18n capabilities, targeting English-proficient Indonesian market
- **Media Management:** Git LFS or external CDN for images, optimized image processing via Hugo
- **SEO Optimization:** Hugo's excellent SEO capabilities with custom templates for meta tags, structured data, and keyword optimization
- **Content Workflow:** Simple Git workflow - write in Markdown, commit, automatic deployment
- **Automation Capability:** n8n integration for automated content generation from spreadsheets to Hugo markdown files
